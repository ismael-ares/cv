  \item Integré herramientas del ecosistema en los pipelines: consulta de tickets Jira vía MCP en procesos de code review y notificaciones a MS Teams mediante Adaptive Cards, incluyendo alertas automáticas de migraciones de base de datos detectadas en PRs.
